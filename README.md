# Personal Portfolio Website

A minimalist single-page portfolio with a vertical timeline. Projects are revealed as Polaroid cards when clicking thumbtacks, with a camera flash effect and photo development animation.

## Features

- Viewfinder cursor (40px square outline)
- Vertical timeline with thumbtacks
- Click interaction: flash â†’ reveal Polaroid
- Photo development animation (blur/brightness fade)
- Minimalist design

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
